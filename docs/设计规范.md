# 财务管理 App 设计规范

**文档版本**：V1.0
**风格**：Apple iOS 18
**适用**：iOS / Android 移动端

---

## 一、设计原则

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| **简洁（Clarity）** | 内容优先，减少视觉干扰 |
| **一致（Consistency）** | 相同的场景使用相同的设计 |
| **层次（Hierarchy）** | 重要信息突出，次要信息退隐 |
| **反馈（Feedback）** | 每个操作都有明确的结果反馈 |

### 1.2 Apple 设计语言特征

- 大量留白，让内容呼吸
- 圆角 everywhere（卡片、按钮、输入框）
- 卡片式布局，信息分组清晰
- 图标简洁，SF Symbols 风格
- 字体层级分明

## 五、圆角规范

### 5.1 圆角尺寸

| 元素 | 圆角值 | 说明 |
|------|--------|------|
| 页面卡片 | 16px | 页面内主要信息卡片 |
| 小卡片 | 12px | 列表项、小型组件 |
| 按钮 | 12px | 主要按钮、次要按钮 |
| 输入框 | 10px | 文本输入框 |
| 标签/徽章 | 8px | 小型标签、状态徽章 |
| 头像/图标 | 50% | 圆形头像、圆形图标 |

### 5.2 圆角使用规则

- 卡片统一使用 16px 圆角
- 按钮统一使用 12px 圆角
- 输入框统一使用 10px 圆角
- 同一页面内的圆角尺寸保持一致

## 七、组件规范

### 7.1 按钮

**主按钮（Primary Button）**
```
高度: 50px
背景: #007AFF (主色)
文字: #FFFFFF, 17pt, Semibold
圆角: 12px
字重: Center
```

**次按钮（Secondary Button）**
```
高度: 50px
背景: #F2F2F7 (背景灰)
文字: #007AFF, 17pt, Semibold
圆角: 12px
```

**文字按钮（Text Button）**
```
高度: 44px (最小触摸区)
文字: #007AFF, 17pt, Regular
无背景
```

### 7.2 输入框

```
高度: 44px
背景: #F2F2F7
边框: 无 (纯色背景风格)
圆角: 10px
内边距: 12px 16px
文字: #000000, 17pt
占位符: #8E8E93
```

### 7.3 卡片

```
背景: #FFFFFF
圆角: 16px
内边距: 16px
阴影: 0 2px 8px rgba(0,0,0,0.08)
边框: 1px solid #E5E5EA (可选)
```

### 7.4 列表项

```
高度: 44px (最小)
背景: #FFFFFF
左侧内边距: 16px
右侧内边距: 16px
底部边框: 1px solid #E5E5EA (最后一项无)
```

### 7.5 底部导航栏

```

背景: #F9F9F9
边框顶部: 0.5px solid #E5E5EA
图标大小: 25x25pt
文字大小: 10pt
选中颜色: #007AFF
未选颜色: #8E8E93
```

## 十一、动画规范

### 11.1 过渡动画

| 场景 | 时长 | 曲线 | 效果 |
|------|------|------|------|
| 页面切换 | 300ms | ease-in-out | 横向滑动 |
| 弹窗出现 | 250ms | ease-out | 从下向上滑入 |
| 弹窗消失 | 200ms | ease-in | 向下滑出 |
| 按钮点击 | 100ms | ease-in | 透明度变化 |

### 11.2 交互动画

| 场景 | 效果 |
|------|------|
| 按钮点击 | 缩放 0.97 + 透明度 0.8 |
| 列表项点击 | 背景色变灰后恢复 |
| 开关切换 | 滑动 + 颜色渐变 |
| 加载中 | 骨架屏闪烁或旋转图标 |

---

## 十二、可访问性

### 12.1 触摸区域

- 最小触摸区域：44x44pt
- 列表项高度：至少 44px
- 按钮内文字与边缘间距：至少 12px

### 12.2 对比度

- 文字与背景对比度：至少 4.5:1
- 重要图标与背景对比度：至少 3:1

### 12.3 字体缩放

- 支持系统字体大小设置
- 避免使用固定像素值
