# UI 原型图生成 Prompt

请为财务管理 App V2.0（语音记账功能）生成以下 UI 原型图。

## 设计风格

- **整体风格**：Apple Design（简洁、统一、一致）
- **配色**：参考 iOS 系统色，蓝色主色调 (#007AFF)
- **圆角**：16-24px 大圆角
- **字体**：SF Pro 风格，无衬线字体
- **图标**：线性图标风格，线条粗细 1.5-2px
