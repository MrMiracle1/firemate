## 设计统一规范 Prompt（供生成所有页面时参考）

```
iOS 18 设计规范要求：
- 字体：SF Pro（正文）、SF Pro Display（标题）
- 圆角：卡片16px、按钮12px、输入框10px
- 间距：页面边距16px、卡片内边距16px、元素间距12px
- 色彩：主色#007AFF(蓝色)、背景#FFFFFF、文字#000000(标题)/#8E8E93(副文)、分割线#E5E5EA
- 阴影：卡片0 2px 8px rgba(0,0,0,0.08)
- 图标：SF Symbols风格，圆形图标背景，24x24pt
- 按钮：圆角矩形，全宽或自适应，高度44pt(最小触摸区域)
- 导航：底部标签栏高度49pt+34pt(安全区域)，顶部标题居中
```
