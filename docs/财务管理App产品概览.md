# 财务管理 App 产品概览文档

**文档版本**：V1.0
**日期**：2026-03-07

---

## 一、产品定位

**产品名称**：（暂定，如"财富通"、"账本精灵"）

**一句话描述**：一款专注于个人财富管理的智能记账工具，通过语音输入高效记账，帮助用户统一管理分散在银行卡、支付宝、微信、股票基金等多渠道的资产，并提供预算控制和 AI 智能分析功能。

**核心愿景**：让用户对自己的钱有足够的掌控力，清醒地看到自己的钱慢慢增值的过程。

---

## 二、核心价值主张

| 痛点 | 解决方案 |
|------|----------|
| 钱太散（多银行卡、支付宝、微信） | 统一资产视图，一个界面看遍所有账户 |
| 手动记账繁琐 | 语音输入 + 智能解析，用户只需确认 |
| 不知道钱花去哪了 | 消费分类 + 预算对比 + AI 月度总结 |
| 资产配置混乱 | 账户分类（保底/增值/储蓄）+ 目标储蓄 |
| 钱不知道该怎么分配 | AI 智能理财建议，工资到账自动分配方案 |
| 看不到钱增值的过程 | 资产增长可视化，清晰看到资产变化趋势 |

---

## 三、产品形态

- **移动端 App**（Android 优先）
- **后台管理系统**（Web 端，用于管理员查看数据）
- **前后端分离架构**，后端部署在云服务器
- **云端数据库**（Supabase）

---

## 四、核心功能模块

### 1. 账户管理

**目的**：统一管理用户所有资金渠道

| 账户类型 | 说明 |
|----------|------|
| 现金 | 手动记录余额 |
| 银行卡 | 储蓄卡、信用卡（显示余额） |
| 第三方支付 | 支付宝、微信钱包 |
| 投资账户 | 股票账户、基金账户 |
| 储蓄账户 | 定期存款、稳健理财 |

- 支持新增/编辑/删除账户
- 支持账户余额手动更新（对账功能）
- 支持账户图标、颜色自定义

**内部转账**：
- 支持账户间转账，选择"转出账户"和"转入账户"
- 转账记录流水，余额自动调整
- 转账不计入收支（收支屏障）

---

### 2. 语音记账

**核心流程**（初步方案，后续可能调整）：

```
用户语音输入 → ASR语音转文字 → AI语义解析 → 生成记账卡片 → 用户确认/补充 → 记账成功
```

**语音输入内容解析**：
- 金额
- 交易方向（收入/支出）
- 账户（从哪个账户扣款/入账）
- 分类（餐饮、交通、购物等）
- 时间（默认当天，可指定）
- 备注（可选）

**解析结果展示**：
- 弹窗卡片展示解析结果
- 未能识别的字段高亮提示用户手动补充
- 用户可修改任意字段后确认

**特殊场景 - 年卡分摊**：
- 购买年卡等大额支出时，可备注"分摊12个月"
- 记录为一笔支出，备注中体现分摊信息

---

### 3. 预算管理

**设计思路**：

- 按月设置总预算
- 可按分类设置子预算（如：餐饮 2000、房租 3000）
- 记录每笔支出时实时显示预算进度
- 超预算时提醒

**位置**：嵌入在"记账"模块中，作为控制阀存在

---

### 4. 资产管理（股票/基金）

- 支持手动添加股票、基金持仓
- 记录买入成本、当前市值
- 计算盈亏金额和收益率
- 归入"投资账户"统一管理

---

### 5. 目标储蓄

- 可设定目标（如：储蓄账户存到 10 万、股票账户达到 20 万）
- 显示目标进度条
- 可关联具体账户

---

### 6. AI 智能理财助手

**模块设计**：独立的"AI 理财助手"模块，作为用户的智能财务顾问

| 功能 | 说明 |
|------|------|
| 资金分配建议 | 手动录入收入后，AI 自动分析用户财务状况，给出每笔钱的分配建议（如：30% 存入保底账户、20% 投入投资账户、50% 用于生活开支） |
| 月度总结 | AI 分析本月收支情况，生成文字报告 |
| 财务建议 | 根据消费习惯给出优化建议 |
| 自定义分析 | 用户可提问，AI 回答（如：我上个月餐饮花了多少？） |
| 资产增长追踪 | 可视化展示资产变化趋势 |

- 联网调用云端 AI API
- 支持历史数据查询和分析

---

### 7. 资产增长可视化

- 资产总览大屏，直观看到总资产变化
- 时间轴展示资产增值过程
- 区分"收入"、"支出"、"投资收益"对资产的贡献
- 让用户清晰地感受到"钱在慢慢变多"

---

### 8. 报表与可视化

- 饼图：支出分类占比
- 折线图：月度收支趋势
- 条形图：各账户余额对比
- AI 灵活呈现 + 预设模板并存

### 9. 首页仪表盘

**核心信息展示**：
- 总资产：用户所有账户余额汇总
- 本月预算剩余：当前月预算使用情况
- 本月收入/支出对比：直观展示月度收支平衡状态

## 五、数据结构设计（概要）

```
用户
  └── 账户（一级）
       └── 交易流水（二级）
            ├── 金额
            ├── 类型（收入/支出/转账）
            ├── 分类
            ├── 时间
            ├── 备注
            └── 转账关联账户

预算
  ├── 月度总预算
  └── 分类预算

目标
  ├── 目标名称
  ├── 目标金额
  └── 关联账户
```

## 六、安全与隐私

- 数据库：云端数据库（Supabase）
- 前后端分离，API 鉴权

## 七、UI 设计方向

- **风格**：参考 Apple 设计语言（简洁、统一、一致）
- **色调**：统一配色，主色 + 辅助色
- **交互**：卡片式设计，圆角，充足的留白
- **图标**：统一风格的 icon set
